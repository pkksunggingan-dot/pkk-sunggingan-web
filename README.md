# Website Resmi TP PKK Kelurahan Sunggingan

Website resmi Tim Penggerak Pemberdayaan dan Kesejahteraan Keluarga (TP PKK) 
Kelurahan Sunggingan, Kecamatan Kota, Kabupaten Kudus, Jawa Tengah.

**🌐 Live:** [pkksunggingan.my.id](https://pkksunggingan.my.id)

---

## 📋 Tentang Project

Website ini dikembangkan sebagai bagian dari program KKN (Kuliah Kerja Nyata) 
mahasiswa Universitas Muria Kudus (UMK) tahun 2026 untuk mendigitalisasi 
informasi dan kegiatan TP PKK Kelurahan Sunggingan.

### Fitur Utama
- 📰 Berita & kegiatan PKK
- 👥 Susunan pengurus (33 anggota)
- 📅 Program kerja 2026 per Pokja
- 🏥 Informasi Posyandu & layanan warga
- ♻️ Program unggulan (Bank Sampah, UP2K, UMKM)
- 📞 Kontak & lokasi sekretariat

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Auth:** Supabase (email + password)
- **CMS:** Sveltia CMS (Git-based)
- **Hosting:** Cloudflare Workers/Pages
- **Domain:** DomaiNesia (.my.id)
- **Email:** Cloudflare Email Routing + Gmail SMTP
